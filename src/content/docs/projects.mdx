### [Grift](https://github.com/skyfskyf/grift)

A vau-calculus Lisp interpreter suitable for bare metal devices and embedded use.
Implements first-class operatives (fexprs) that subsume both functions and macros,
with a fixed-size arena allocator, mark-and-sweep GC, and tail-call optimization.

<span class="tag">Rust</span>
<span class="tag">no_std</span>
<span class="tag">Lisp</span>
<span class="tag">Vau Calculus</span>

**Features:**
- `no_std`, `no_alloc`, `#![forbid(unsafe_code)]`
- Fixed-size arena with free-list allocation
- First-class operatives (fexprs) via vau calculus
- Tail-call optimization via trampoline
- Mark-and-sweep garbage collection
- Checked integer arithmetic

### [bevy_ratatui](https://github.com/ratatui/bevy_ratatui)

Set up a Ratatui terminal UI application using Bevy to manage the update loop,
handle input events, and draw to the buffer. Supports both terminal and windowed
rendering modes.

<span class="tag">Rust</span>
<span class="tag">Bevy</span>
<span class="tag">Ratatui</span>
<span class="tag">TUI</span>

**Features:**
- Bevy-powered update loop for terminal applications
- Crossterm input event forwarding
- Optional windowed rendering mode
- WASM-compatible for browser deployment
