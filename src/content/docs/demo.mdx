## About This Demo

This REPL is a **bevy_ratatui** application compiled to WebAssembly. It runs the
full Grift interpreter directly in your browser вҖ” no server required. The terminal
UI is rendered using [Ratatui](https://ratatui.rs) via the
[bevy_ratatui](https://github.com/ratatui/bevy_ratatui) windowed backend, with
[Bevy](https://bevyengine.org) managing the update loop and input handling.

### Try These Examples

```lisp
;; Basic arithmetic
(+ 1 2 3)              ; вҶ’ 6
(* 2 (+ 3 4))          ; вҶ’ 14

;; Define variables and functions
(define! x 42)
(define! double (lambda (n) (* n 2)))
(double x)              ; вҶ’ 84

;; Vau operatives вҖ” the core of Grift
(define! my-quote (vau (x) #ignore x))
(my-quote (+ 1 2))     ; вҶ’ (+ 1 2)

;; Lists and pairs
(cons 1 (cons 2 (cons 3 ())))  ; вҶ’ (1 2 3)
(car '(a b c))                  ; вҶ’ a
(cdr '(a b c))                  ; вҶ’ (b c)

;; Tail-recursive fibonacci
(define! fib (lambda (n a b)
  (if (= n 0) a
    (fib (- n 1) b (+ a b)))))
(fib 50 0 1)            ; вҶ’ 12586269025

;; Higher-order functions
(define! map (lambda (f lst)
  (if (null? lst) ()
    (cons (f (car lst)) (map f (cdr lst))))))
(map double '(1 2 3 4))  ; вҶ’ (2 4 6 8)
```

### Building from Source

To compile the REPL WASM module from source:

```bash
# Clone the grift repository
git clone https://github.com/skyfskyf/grift.git
cd grift

# Build the WASM REPL (requires wasm-pack)
wasm-pack build --target web crates/grift-repl

# Copy the output to grift-site/public/wasm/
cp pkg/* ../grift-site/public/wasm/
```
